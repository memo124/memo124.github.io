Sistema de tienda en linea

Sistema de uso interno para la compra de articulos de automoviles, tanto la gestión de salida de articulos como el control de reseñas hechas por los clientes.

Repositorio de GitHub

Descripción del proyecto

Proyecto estudiantil, para el desarrollo de una tienda en linea, tanto de uso para los clientes y uso interno para la empresa.

Se dividio en dos partes el sistema realizado, un lado publico donde se muestran los productos disponibles, las reseñas que dejaron los clientes y la calificación que le proporcionaban.
